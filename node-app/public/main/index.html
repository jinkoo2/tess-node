MYOCR.NET

Image-To-Text RESTFUL API

MYOCR.NET is a restful API. Developers can use the API to extract words from images and convert to searchable pdf documents. Given an image, the API returns a structured JSON object with words and their sementic locations in the text. The API is free to use for any applications.
